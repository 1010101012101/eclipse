Refactoring

  • Rename
    • annotations, annotation members (especially the special case value), enum, enum member, type parameters, vararg parameters
    • with varargs, type parameters, generic & parameterized types in the signature (field, method, local)
  • Move CU/Type
    • annotations, enum, generic & parameterized types
  • Move (method & static members)
    • enums and annotations as members in types
    • signature with type parameters, varargs, parameterized types
    • methods in generic & parameterized types
    • methods that refer to a static import or type parameters from class
  • Change method signature
    • signature with type parameters, varargs, parameterized types
    • methods in annotations, enums, generic & parameterized types
  • Convert Anonymous to nested
    • annotations, enum, generic & parameterized types
  • Move member type to new file
    • annotations, enum, generic & parameterized types
    • code that refers to static imports
    • code that refers to type parameters
  • Pull Up & PushDown
    • methods in generic & parameterized types
    • methods that refer to a static import
    • methods with type parameters, varargs, parameterized types
    • between generic, normal and parameterized types
    • enums and annotations as members in types
  • Extract Interface / Use Supertype where possible
    • enum and annotations
    • generic and parameterized types
    • method with varargs, type parameters, parameterized types
  • Generalize Type
    • enums and annotations
    • generic and parameterized types
  • Extract/Inline Method
    • Code in generic & parameterized types and enums
    • Code that refer to type parameters (from class and method)
    • vararg usages
    • code with references to static imports
    • code with autoboxing
  • Extract/Inline local
    • expression which resolve to parameterized types, annotations, enums  and type parameters (or the concrete type for parameterized types)
  • Extract/Inline constant
    • in enums, annotations, generic and parameterized types
  • Introduce parameter
    • expression which resolve to parameterized types, annotations, enums  and type parameters (or the concrete type for parameterized types)
    • signature to be changed contains varargs, type parameters, parameterized types
  • Introduce Factory
    • for generic and parameterized types
    • destination type is generic or parameterized
  • Convert local variable to field
    • declaring type is parameterized types, annotations, enums  and type parameters (or the concrete type for parameterized types)
  • Self encapsulate field
    • field type is parameterized types, annotations, enums  and type parameter
    • in enums, parameterized types and generic types.
  • Infer type arguments
    • test for collection classes
    • test for own generic classes
    • test for classes with n type parameters
