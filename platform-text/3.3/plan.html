Overall Goals

This section defines the goals for Eclipse 3.3 for the Platform Text and the JDT Text component. Each goal is annotated with the overall 3.3 Eclipse theme (see Eclipse Project Draft 3.3 Plan for details).

Platform

  • Increase correctness test coverage of the text infrastructure. Besides providing tests for new features we need to add more tests for existing areas where we have only few or no tests. [Scaling up]
