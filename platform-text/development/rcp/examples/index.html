Eclipse Rich Client Platform - Text Infrastructure

The rich client platform (RCP) is a major theme for Eclipse 3.0. As many applications provide some sort of text editing, an optional RCP text editor component is provided.

For more information about the RCP see the RCP main page, the Rich Client Platform FAQ and the Rich Client Tutorial.

Optional Text Editing Component

Applications that want to offer text editing capabilities may include the following eclipse plug-ins which are an optional part of RCP (i.e. they are not included in the RCP drop available from the Eclipse download site):
  • org.eclipse.text
  • org.eclipse.jface.text
  • org.eclipse.ui.workbench.texteditor

Text Editor Example

The RCP Text Editor Example is an example of a simple RCP app that makes use of the text editing infrastructure (an optional RCP component).
Click here for the documentation for the example.

To run the example in binary form (last updated June 25, 2004 for the 3.0RC4 build):

  1. Download the RCP Runtime build from the Downloads page, and extract it to a folder such as c:\texteditor-example.
  2. Download the RCP text editing component zip and extract it to the eclipse/plugins subfolder.
  3. Download the texteditor plugin zip and extract it to the eclipse/plugins subfolder.
  4. Replace the eclipse/configuration/config.ini file with the appropriate config file from the texteditor plugin. For example, on Windows, copy eclipse/plugins/org.eclipse.ui.examples.rcp.texteditor_1.0.0/config_win32.ini to eclipse/configuration/config.ini .
  5. Run the eclipse executable: For example, on Windows, run eclipse.exe.

To load the text editor example source into your workspace:

  1. Download the texteditor source plugin and save it on your harddisk.
  2. Create a new java project in your workspace and name it org.eclipse.ui.examples.rcp.texteditor. Choose to "Create separate source and output folders".
  3. Choose File > Import > Zip File.
  4. Use the Browse button to select the zip file you downloaded in step 1. Choose the plugin created in step 2 as destination folder.
  5. Press Finish.
To run the example from your workspace:
  1. Choose Run > Debug...
  2. Under Run-time Workbench, there should be two new configurations: RCP Text Editor (gtk) and RCP Text Editor (win32).
  3. Edit the appropriate one as follows:
    • on the Arguments tab, change the workspace location to some directory with write access (different than your host workspace, and different than the workspace used for other launch configs);
    • in the Program to Run group, ensure that the application org.eclipse.ui.workbench.rcp.texteditor.application is selected;
    • on the Plug-ins tab, press Deselect All, check org.eclipse.ui.workbench.rcp.texteditor, press Add Required Plug-ins, and also check org.eclipse.update.configurator.
  4. Press Debug.
  5. When the RCP Text Editor Example window opens, use File > Open to open a plain text, .xml, or .html file.
