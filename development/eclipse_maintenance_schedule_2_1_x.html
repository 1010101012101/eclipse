Eclipse Project
2.1 Maintenance Release Schedule

Last revised March 17, 2004 ( marks interesting recent changes)

Eclipse 2.1 (plan) is the current major release of the Eclipse Project. We plan to fix critical "stop ship" defects in 2.1 and periodically produce 2.1.x maintenance releases incorporating the accumulated fixes. Non-critical problems will generally be addressed only in the next major release of Eclipse (3.0), which is the principal focus of the Eclipse development teams.

The maintenance schedule for Eclipse 2.1 is as follows:

Version Release date Description
2.1.0 March 27, 2003 initial release (release notes)
2.1.1 June 27, 2003 maintenance release fixing critical "stop ship" defects (release notes)
2.1.2 November 4, 2003 maintenance release fixing critical "stop ship" defects in 2.1.1 (release notes)
2.1.3 March 17, 2004 maintenance release fixing critical "stop ship" defects in 2.1.2 (release notes)

Once Eclipse 3.0 ships (2Q2004; see plan), it's unlikely that there would be any call for additional 2.1.x releases.
