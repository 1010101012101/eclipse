Eclipse Project IP Log (1.0 to 3.4)

This document is the IP log for the Eclipse project that was manually maintained prior to the introduction of the automatic IP log tool. This IP data covers Eclipse Project releases 1.0 to 3.4 inclusive.

Section 1 (Committers)

This is a list of the dev.eclipse.org unix login names for each of the Committers, past and present, whose contribution is still active in the repository.

Click here for the list of committers.

Section 2 (Contributers)

A list of all the non-Committers whose contribution is still active in the repository. The columns for this section are:

  • component (CVS directory)
  • bug #
  • contributor's name
  • contribution size
  • committer who committed the contribution
  • description

Click here for the list of contributors.

Section 3 (Third Party Software)

A list of the non-Eclipse third party software included in the Project. The columns for this section are:

  • name including version
  • IPzilla # of entry providing legal clearance for inclusion
  • directory location or jar file
  • license name and version (including any licenses related to embedded third party software)
  • usage (e.g. modified/unmodified, source, object, derivative work, entire package or which subset)

Click here for the list of third party software used in the Eclipse project.
