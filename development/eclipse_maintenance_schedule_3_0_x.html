Eclipse Project
3.0 Maintenance Release Schedule

Last revised June 29, 2005 ( marks interesting recent changes)

Eclipse 3.0 (plan) was the previous major release of the Eclipse Project. We plan to fix critical "stop ship" defects in 3.0 and periodically produce 3.0.x maintenance releases incorporating the accumulated fixes. Non-critical problems are generally addressed only in the next major release of Eclipse (3.1).

The maintenance schedule for Eclipse 3.0.x is as follows:

Version Release date Description
3.0.0 June 25, 2004 initial release (release notes)
3.0.1 Sept. 17, 2004 maintenance release fixing critical "stop ship" defects in 3.0.0 (release notes)
3.0.2 March 23, 2005 maintenance release fixing critical "stop ship" defects in 3.0.1 (release notes)

Since Eclipse 3.1 has shipped, it's unlikely that there will be additional 3.0.x releases.
