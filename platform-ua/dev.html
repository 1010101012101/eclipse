  • org.eclipse.help.ide
  • org.eclipse.help.ui
  • org.eclipse.help.webapp
  • org.eclipse.tomcat
  • org.apache.lucene
  • org.eclipse.ui.cheatsheets

Mailing Lists
  • platform-help-dev@eclipse.org (post, archives)
    Technical questions and discussions about using User Assistance Component, or developing documentation plug-ins should be posted to the newsgroups. Mailing lists at eclipse.org are intended for use by developers actually working on or otherwise contributing to day-to-day development.
