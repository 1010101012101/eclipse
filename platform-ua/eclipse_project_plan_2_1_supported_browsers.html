Supported Browsers in Eclipse 2.1

The Help system will support "out of the box" the following browsers as its external viewer. Additional browsers can be configured into Eclipse based on browser adaptor availability, or browser executable specified in help preferences.

Note: the windowing system, Java platform, operating system and level of testing corresponds to the base Eclipse 2.1 plan.
