Plug-in Customization in Eclipse 2.1

The Eclipse Help page describing how to customize your plug-in.

Plug-in Preference load order:

  1. Load saved preferences from disk.
    - c:/eclipse/workspace/.metadata/.plugins/org.eclipse.core.resources/prefs.ini
  2. Call Plugin#initializeDefaultPluginPreferences().
    - method implented by subclasses of org.eclipse.core.runtime.Plugin
  3. Load defaults from the preferences.ini file in the plug-in's installation directory.
    - c:/eclipse/plugins/org.eclipse.core.resources_2.0.0/preferences.ini
  4. Load defaults from the plugin_customization.ini file in the primary feature's plug-in directory.
    - c:/eclipse/plugins/org.eclipse.platform_2.0.0/plugin_customization.ini
  5. Load defaults from the file specified after the -pluginCustomization command-line argument.
