OSGi

The OSGi framework specification forms the basis of the Eclipse runtime. As of Eclipse 3.0, the runtime is fully based on the OSGi notions of bundle (equivalent to Eclipse plug-ins). The OSGi framework implementation in Eclipse 3.0 is fully compliant with the OSGi framework specification R3.0. For more information on OSGi, see the web site at osgi.org.
