Feature Testing

Input Handling

  • Same JRE
    • with / without valid args
    • cancel
    • -noinput
  • Separate JRE
    • with / without valid args
    • cancel
    • -noinput

Folding Options

  • Make use of the editor context ruler menu
  • Key binding actions: Expand, Expand All, Collapse, Enable folding
  • Folding is presented for all the partitions in the Ant buildfile. Partion hover is correct.

Hyperlink Navigation

  • Enable / disable
  • Key Modifier and modifier manipulation (preference page)
  • Links presented for all reasonable Ant elements

External Documentation

  • Works for all Ant elements
    • tasks
    • projects
    • targets
    • types
  • Adapts to key binding change

AntView

  • No loss of functionality from 3.0
    • Context menu actions
    • DnD
    • Add/removal
    • Search
    • Running targets (default and otherwise)
  • New functionality from 3.0
    • Navigation to targets with selection for normal, external entities and imported targets

Navigation

  • Open declaration support works for
    • properties (from files as well)
    • targets
    • referenced elements

Import Project from Buildfile

  • Create a Project using an external buildfile containing a javac task
    • error cases
    • correct links setup for multi-platform
    • correct links for various destdir and sourcedir
    • correct links for various destdir and sourcedir
